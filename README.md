# Policies

This repository is configured for GitHub Pages + Jekyll so Markdown policy files are published as web pages.

## Available Policies

1. [CTViz Privacy Policy](app-privacy-policies/ctviz/policy.md)

## Subject Areas

- [App Privacy Policies](app-privacy-policies/README.md)
- [Respiratory Therapy](respiratory-therapy/README.md)


## Published Site URL

When GitHub Pages is enabled for this repository as a **project site**, the base URL is:

- `https://zbzfirst.github.io/Policies/`

Example policy URL:

- `https://zbzfirst.github.io/Policies/app-privacy-policies/ctviz/privacy-policy/`

## Structure

- `index.md` — landing page listing available policies.
- `app-privacy-policies/` — subject folder for application privacy policies.
  - `ctviz/policy.md` — CTViz policy body content file.
- `respiratory-therapy/` — subject folder for Respiratory Therapy policies and evidence.
  - `policies/` — policy documents.
  - `supporting-evidence/` — supporting articles/guidelines.
- `_config.yml` — minimal Jekyll site configuration.
